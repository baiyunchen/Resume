系统内对外提供数据的接口，尽量遵循Restful风格。该项目是基于公司内部API框架而建，并没有使用微软MVC等框架，整理项目架构为简单三层，实现了多语言、SQL代码可配置。

项目上并没有太多可以说的点，该项目中，主要的挑战来自于数据库数，由于数据量较大，且组内相关数据操作大部分为批量操作，所以日常代码在编写过程中，对SQL性能要求较高。
